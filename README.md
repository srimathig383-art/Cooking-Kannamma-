# 🍲👗 Cooking Kannamma Collections

A modern, responsive e-commerce website for showcasing homemade food items and dress collections with WhatsApp ordering.

---

## 🚀 Live Website
👉 https://srimathig383-art.github.io

## 🌟 Features

- 🛒 Add to Cart System  
- 📲 WhatsApp Order Integration  
- 📸 Image Gallery (Easy to update)  
- 🔍 Image Zoom on Click  
- 🎨 Instagram-style UI Design  
- 📦 Order Summary via WhatsApp  
- ⭐ Customer Reviews Section  
- 📱 Fully Mobile-Friendly
