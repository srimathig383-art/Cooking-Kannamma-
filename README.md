# 🍲👗 Cooking Kannamma Collections

A modern and responsive e-commerce website for homemade food items and dress collections.

## 🌟 Features

- 🛒 Add to Cart System
- 📲 WhatsApp Order Integration
- 📸 Image Gallery (Easy to update)
- 🔍 Image Zoom on Click
- 🎨 Instagram-style UI Design
- 📦 Order Summary via WhatsApp
- ⭐ Customer Reviews Section
- 📱 Mobile-Friendly Website
